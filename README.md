This repository contains the dataset used in the "2T-XARBI" study (by Sibel Gulmez, Benjamin C. M. Fung, Arzu Gorgulu Kakisim, Ibrahim Sogukpinar, Ching-Chun Huang, and Shih-Chia Huang).

The dataset is organized into folders based on ransomware families. For each sample, two files are provided: a .behavior file and a .json file. The .behavior file contains the behavior labels associated with the sample. The .json file represents the assembly code of the sample in a graph structure, as described in the paper. Both files are named using the hash of the corresponding executable. The executable files can be retrieved from MalwareBazaar using these hashes.
